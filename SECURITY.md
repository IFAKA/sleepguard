# Security Policy

SleepGuard is intentionally local-first. It does not collect analytics, send network requests, or store secrets.

## Reporting

Please open a private report or contact the maintainer before publicly disclosing security issues.

## Scope

Useful reports include:

- Unsafe shell execution.
- Privilege escalation risks.
- LaunchAgent persistence that exceeds the documented user schedule.
- Behavior that blocks normal macOS unsaved-work prompts.

Out of scope:

- Requests to make SleepGuard harder to bypass.
- Parental-control or device-management enforcement features.
